## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Project Overview

Build a fully functional, production-ready SaaS platform called **eTopia** that empowers communities through digital inclusion, social innovation, entrepreneurship, workforce development, and collaborative problem solving.

The platform operates as a hybrid ecosystem consisting of a Commercial Track and a Social Track that together create a self-sustaining digital society capable of addressing local and global challenges.

eTopia must be designed for NGOs, governments, universities, enterprises, startups, social entrepreneurs, volunteers, investors, mentors, and underserved communities.

The system should be enterprise-grade, cloud-native, AI-powered, multilingual, mobile-first, highly scalable, and ready for deployment.
